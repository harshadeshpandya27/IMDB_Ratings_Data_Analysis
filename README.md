# 🎬 IMDB Ratings Analysis & Interactive Dashboard

## 📌 Overview
This project analyzes IMDB movie data to uncover patterns in ratings, audience preferences, and genre performance. It combines Python-based data analysis with an interactive Power BI dashboard to enable intuitive exploration of insights.

## 🎯 Objective
To identify key factors that influence movie ratings and audience engagement, helping stakeholders make data-driven decisions in content strategy and production.

## 🚀 Key Insights
- Movies with higher vote counts tend to have more reliable and consistent ratings  
- Genres like Drama and Biography consistently achieve higher average ratings  
- Recent movies show increased audience engagement compared to older releases  
- No strong correlation between movie duration and rating quality  

## 📊 Interactive Dashboard (Power BI)
The Power BI dashboard provides a dynamic way to explore the dataset:

- 🎯 Filter movies by **genre, year, and rating range**
- 📈 Visualize **rating distribution and trends over time**
- 🔍 Identify **top-performing movies and genres**
- 📊 Compare **audience engagement using vote counts**

> The dashboard transforms raw data into actionable insights for quick decision-making.

## 🛠️ Tech Stack
- Python (Pandas, NumPy)
- Data Visualization: Matplotlib, Seaborn
- Power BI (Dashboard & Data Storytelling)
- Jupyter Notebook

## 📂 Dataset
- Source: IMDB dataset  
- Size: 10,000+ records  
- Key Features: Title, Genre, Rating, Votes, Year, Duration  

## 🔍 Approach
- Cleaned and preprocessed raw data (handled missing values, corrected inconsistencies)
- Performed exploratory data analysis (EDA) to uncover trends and relationships
- Built visualizations to communicate insights effectively
- Developed an interactive Power BI dashboard for end-user exploration

## 📈 Business Impact
- Helps production teams identify **high-performing genres**
- Enables better understanding of **audience preferences**
- Supports **data-driven decision-making** in content planning
- Demonstrates how analytics can bridge raw data and business strategy

## ▶️ How to Run
1. Clone the repository  
2. Install dependencies:  
   `pip install pandas numpy matplotlib seaborn`  
3. Run the Jupyter Notebook  
4. Open the Power BI `.pbix` file to explore the dashboard
